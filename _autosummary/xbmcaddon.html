xbmcaddon¶

A class to access addon properties

Classes

Addon([id]) Creates a new Addon class.
class xbmcaddon.Addon(id=None)¶

Bases: object

Creates a new Addon class.

Parameters:id – string - id of the addon (autodetected in XBMC Eden)

Example:

self.Addon = xbmcaddon.Addon(id='script.recentlyadded')
getLocalizedString(id)¶

Returns an addon’s localized ‘unicode string’.

Parameters:id – integer - id# for string you want to localize.

Example:

locstr = self.Addon.getLocalizedString(id=6)
getSetting(id)¶

Returns the value of a setting as a unicode string.

Parameters:id – string - id of the setting that the module needs to access.

Example:

apikey = self.Addon.getSetting('apikey')
setSetting(id, value)¶

Sets a script setting.

Parameters:
  • id – string - id of the setting that the module needs to access.
  • value – string or unicode - value of the setting.

Example:

self.Settings.setSetting(id='username', value='teamxbmc')
openSettings()¶

Opens this scripts settings dialog.

getAddonInfo(id)¶

Returns the value of an addon property as a string.

Parameters:id – string - id of the property that the module needs to access.

Note

Choices are (author, changelog, description, disclaimer, fanart, icon, id, name, path profile, stars, summary, type, version)

Example:

version = self.Addon.getAddonInfo('version')
